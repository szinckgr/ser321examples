You can make the following GET requests:

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a JSON response of the /random request
  • /random -- returns index.html
  • /convert?value=25&unit=celsius - Converts temperature (Celsius to Fahrenheit) (Test)
  • /reverse?string=hello - Reverses the input string (Test)
  • /distance?x1=0&y1=0&x2=3&y2=4 - Calculates the Euclidean distance between two points (Test)
